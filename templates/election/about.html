About Our Platform

This online voting platform was developed as part of an exam at Adventist University of Central Africa (AUCA). The platform allows users to participate in secure, reliable, and transparent elections. It ensures that each user can vote only once in each election, and real-time statistics are displayed to monitor the voting progress.

The main goal of this project is to showcase the application of web technologies and API services learned during the course, and to provide a practical solution for conducting elections in an efficient and user-friendly way.
